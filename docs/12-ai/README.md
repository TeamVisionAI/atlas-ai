# AI Documentation

## AI Summary

The `/docs/12-ai/` section is Atlas's canonical guidance for AI assistants and human operators using the Knowledge Hub. It defines how agents should read documentation, follow prompts, respect business rules, and stay aligned with GitHub as the single source of truth. Documents here are placeholders to expand as Atlas Copilot matures.

## Purpose

Centralize AI-oriented documentation: guidelines, prompts, conversation patterns, Knowledge Hub usage, and development context for Cursor, Copilot, and future Atlas agents.

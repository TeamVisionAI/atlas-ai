# Atlas Request for Comments (RFCs)

**Status:** FROZEN (Version 1)  
**Last Updated:** 2026-07-21  

RFCs define **permanent contracts** for Atlas Version 1. They specify interfaces and principles — not implementation details.

Implementation may change; RFCs should not change without a Version 2 review.

| RFC | Title | Scope |
|-----|-------|-------|
| [RFC-001](./RFC-001-message-envelope.md) | Message Envelope | Gateway internal message contract |
| [RFC-002](./RFC-002-event-naming.md) | Event Naming | Event Bus naming conventions |
| [RFC-003](./RFC-003-tool-contract.md) | Tool Contract | Agent tool request/result schema |
| [RFC-004](./RFC-004-workflow-contract.md) | Workflow Contract | Workflow intelligence contract |
| [RFC-005](./RFC-005-package-manifest.md) | Package Manifest | Package registration and isolation |
| [RFC-006](./RFC-006-organization-model.md) | Organization Model | Organization Console schema |
| [RFC-007](./RFC-007-connector-contract.md) | Connector Contract | Production connector interface |
| [RFC-008](./RFC-008-daily-brief-schema.md) | Daily Brief Schema | Daily Brief document structure |
| [RFC-009](./RFC-009-mission-control-state.md) | Mission Control State | Live operational state schema |
| [RFC-010](./RFC-010-event-bus-principles.md) | Event Bus Principles | Pub/sub architecture rules |
